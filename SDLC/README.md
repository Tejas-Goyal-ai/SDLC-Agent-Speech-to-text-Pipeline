# SDLC Agent

An AI-powered SDLC assistant that captures meeting audio, detects speech using Silero VAD, and will later generate requirements, user stories, BRDs, and knowledge graphs.

## Current Features

- Google Meet Audio Capture
- Microphone Capture
- Audio Mixing
- PCM Streaming via WebSocket
- FastAPI Backend
- Silero Voice Activity Detection

## Tech Stack

- HTML + JavaScript
- Web Audio API
- FastAPI
- WebSockets
- PyTorch
- Silero VAD
